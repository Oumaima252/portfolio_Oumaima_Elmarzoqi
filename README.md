# Portfolio Personnel

##  Description
Ce projet est un site web de portfolio personnel développé avec HTML, CSS et JavaScript.

Il présente mon profil, mes compétences et mes projets avec des fonctionnalités interactives et un design responsive.

---

##  Fonctionnalités
- Design responsive (mobile et desktop)
- Menu de navigation dynamique
- Scroll fluide entre les sections
- Fenêtre modale pour les projets
- Formulaire de contact avec validation
- Chargement de données avec Fetch API

---

##  Technologies utilisées
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Fetch API
- GitHub Pages
