# Portfolio Personnel

##  Description
Ce projet est un site web de portfolio personnel développé avec HTML, CSS et JavaScript.

Il permet de présenter mon profil, mes compétences et mes projets avec des fonctionnalités interactives.

---

##  Fonctionnalités
- Design responsive (mobile et desktop)
- Menu de navigation dynamique
- Fenêtre modale pour les projets
- Validation du formulaire de contact
- Chargement de données avec Fetch API

---

## Technologies utilisées
- HTML5
- CSS3
- JavaScript 
- Fetch API
